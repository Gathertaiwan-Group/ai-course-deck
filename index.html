SECURITY FIRST

上線後，先換 key

所有貼給 AI 的 token 與 API key，上線後都要重新產生。

01 撤銷舊 key

確認舊憑證立即失效。

02 更新 Vercel

把新值放進環境變數，再重新部署。

03 私密憑證不上前端

private key 與 Supabase service-role key 絕不可放進前端。
